WELCOME TO WEB DEVELOPMENT

# TECHNOLOGIES THAT DRIVE THE WEB :

  1. HTML (Hypertext Markup Language)
    • Hypertext: Text that contains links to other text
    • Markup: Wraps the content in markups/ tags to perform various operations ont it
    • Language: It has it's own syntax i.e. it has a right and wrong way of writing it.
    • It provides the structure to our content, annotates it and defines the structure.
  2. CSS (Cascading Style Sheets)
    • It provides the colorinf and styling to our content.
  3. JavaScript
    • This is the programming language that provides a proper behaviour and functionality to our content.

LET US START WITH HTML...

HTML follows sequential rendering i.e. top to bottom

Prior to HTML5 specification , HTML elements were either Block-level Elements or Inline elements; HTML5 split these two content models into 7 models.All the models lie under the given two models:

  1. BLOCK-LEVEL ELEMENTS (roughly flow content)
    • Render to begin on a new line.
    • May contain inline or other block level elements.
  2. INLINE ELEMENTS (roughly phrasing content)
    • Render on the same line by default.
    • May conatin only inline elements.
